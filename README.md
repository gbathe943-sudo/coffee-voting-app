# ☕ Coffee Rating Application

A web application built using Python, Flask, HTML, CSS, and SQLite that allows users to vote for their favorite coffee.

## Features

- View coffee menu
- Vote for coffee items
- Real-time vote updates
- SQLite database integration
- Simple and responsive interface

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
